# GXMicro Network Interface Card drivers
GXMicro Network Interface Card drivers

# 支持内核版本
| 版本 |
| :---: |
| 5.4.18 |
| 6.2.16 |
| 6.5.13 |

# 目录
| 目录 | 驱动目录 |
| :---: | :---: |
| mqnic | 开源 10G (https://github.com/corundum/corundum.git) |
| mfd | Multifunction (drivers/mfd) |
| ata | SATA (drivers/ata) |
| dma | DMA (drivers/dma) |
| spi | SPI (drivers/spi) |
| i2c | I2C (drivers/i2c/busses) |
| serial | Serial (drivers/tty/serial) |
